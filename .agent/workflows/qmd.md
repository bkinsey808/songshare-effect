---
description: Search for relevant skills and documents using QMD
---

This workflow helps you find the most relevant `skills/` and `docs/` files for a task.

1. Run the search command with your task description:
// turbo
```bash
npm run qmd -- search "$ARGUMENTS"
```

2. Review the results and load the most relevant files using `view_file`.
3. If the results are poor, try more specific technical keywords or use the `--all` flag.
